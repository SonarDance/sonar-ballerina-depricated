# SonarQube Ballerina Plugin (Sonar Plugin API)

## How it works

This Plugin provides Ballerina static code analysis support for SonarQube.

- With tests

```cmd
gradlew clean build
```

- Ignoring tests

```cmd
gradlew clean build -x test
```
